Exploring IoT with Cisco Packet Tracer

This course teaches how to set up a smart home network using Cisco Packet Tracer, how to add IoT devices to a home network, and then connect, configure, and monitor them. Learners become more familiar with IoT devices and how they can be programmed. Learners will create their own IoT device and use it in a simulated smart home network.

Course Benefits

Practice IoT (Internet of Things) skills using Cisco Packet Tracer.

Explores Opportunities in Technology

✓ Connect and monitor devices in a smart home network.
✓ Modify and monitor environment elements that may affect IoT devices.
✓ Create and modify an IoT Thing in Cisco Packet Tracer.
