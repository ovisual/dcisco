CCNP Enterprise: Core Networking (ENCOR)

This first course in the 2-course CCNP Enterprise series covers switching, routing, wireless, and related security topics, along with the technologies that support software-defined, programmable networks.

Course Benefits

Gain practical, hands-on experience and skills needed to configure, operate and troubleshoot large scale enterprise networks.

Prepares for Careers

✓ Develop skills for professional-level networking roles
✓ Prepare for the Cisco Enterprise Network Core Technologies exam (350-401 ENCOR) to earn an Enterprise Core Specialist certification
✓ Completion of both CCNP Enterprise courses prepares for CCNP Enterprise certification
